# CalendarioCumple
Calendario de cumpleaños
El propósito de esta aplicación es implementar todas las fechas de cumpleaños de tus amigos o familiares para que haya un registro de todos esos cumpleaños y que no se llegue a olvidar la felicitación de dicho cumpleaños.
La aplicación es simple ya que consta de un título de aplicación, una imagen para darle un poco de vida a la misma, de dos cuadros de texto para ingresar las fechas de cumpleaños con sus nombres y su respectivo botón para añadir estas fechas y en la parte inferior de la aplicación se nos muestran las próximas fechas de cumpleaños.
